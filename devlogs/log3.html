Dev Log 3: The Start of Digital Prototyping

01/23/20232 min read

This week, we began digital prototyping for Lumina. The focus was to set up the development environment and project in Unity and to have some assets drawn out.

In this first image, you can see a prototype of how the atmosphere may look along with the player character.

Here are digital mockups of the player character while using the five different abilities. We have the blue dash, orange glide, pink shield, green growth, and yellow double jump abilities.

With these mockups, we began work on Luminaâ€™s key visual and logo, which we will have next week. As digital development progresses, we will begin to have proper assets for the game.

Furthermore, we were able to start learning about Unity through various 2D platformer tutorials. Setting up the project and getting it shared through version control was a bit time consuming, but we are ready to begin development soon.

From here, we will be beginning our digital development in Unity to have a minimum viable product (MVP) by the end of next week. We hope that our MVP will have one level which allows the player to use a couple abilities along with some background music.
